Project Overview

This project explores various deep learning and machine learning approaches to predict cryptocurrency price movements, specifically focusing on Bitcoin (BTC) and Ethereum (ETH).

Multiple Model Architectures

Compares RNN, GRU, CNN, and Transformer architectures

Data Percentage Analysis

Tests models on different dataset sizes (0.1%, 1%, 10%)

Multi-Asset Support

Applies models to both Bitcoin and Ethereum

Backtesting Framework

Evaluates trading performance with stop-loss protection
